# 朴素贝叶斯算法
## 贝叶斯公式
### P(A|B)=P(B|A)*P(A)/P(B)
* P(A|B)：在B发生的前提下，A发生的概率
* P(B|A)：在A发生的前提下，B发生的概率
* P(A):A发生的概率
* P(B):B发生的概率
### 解释
* A:分类
* B:特征
### 在实际应用场景中
* A代表分类
* B代表分词
* P(A|B):包含B分词的文章是A分类的概率
* P(B|A):是A分类的文章中，包含B分词的概率
* P(A):A分类文章的概率
* P(B):包含B分词的概率
* P(A|B)和P(B|A)很难统计，所以可以通过是分类和不是分类的比做为参考K=P(A|B)/P(非A|B),
约分掉不好计算的参数，K越大越可能是某篇文章
